LESSON 12: TYPE ALIASSES VS INTERFACE
  - type cannot use reopening of interface
  - interface has a differnt approach in inheritance

// TYPE ALIASES inheritance
type Animal = {
  name: string;
}

type Bear = Animal & { 
  honey: boolean;
}

// INTERFACE inheritance
interface Animal {
  name: string;
}

interface Bear extends Animal {
  honey: boolean;
}

Docs: https://www.typescriptlang.org/docs/handbook/2/everyday-types.html#differences-between-type-aliases-and-interfaces\
